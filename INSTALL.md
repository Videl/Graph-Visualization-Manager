INSTALL
=======

Requirements
------------
 * Graphviz
     *  ``sudo apt-get install graphviz``
     *  ``cd /usr/ports/graphics/graphviz && make install``
     *  ``port install graphviz``
 * GCC or Clang
 * make
 * any image reader such as ``feh``
 
 Compile
 -------
 
 ``
 make
 ``
 
 Launch
 ------
 ``
 ./general
 ``
