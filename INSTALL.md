INSTALL
=======

Requirements
------------
 * Graphviz
   ''sudo apt-get install graphviz'' or ''cd /usr/ports/graphics/graphviz && make install'' or ''port install graphviz''
 * GCC or Clang
 * make
 * any image reader such as ''feh''
 
 Compile
 -------
 
 ``
 make
 ``
 
 Launch
 ------
 ``
 ./general
 ``
